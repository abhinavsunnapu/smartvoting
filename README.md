# Smart Voting System

A full-stack digital voting platform built using React.js, Node.js, Express.js, and MongoDB.

## Features

- JWT Authentication
- OTP Verification
- Role-Based Access Control
- Secure Vote Storage
- Real-Time Election Dashboard
- Admin & Voter Panels

## Tech Stack

Frontend:
- React.js
- CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB

## Installation

### Frontend

npm install
npm start

### Backend

npm install
npm start
